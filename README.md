# 🚗 Road Accident Analysis using Python

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an Indian Road Accident dataset containing **20,000 accident records** and **24 features**. The objective is to identify accident patterns, analyze contributing factors, and generate insights that can support road safety and decision-making.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Analyze accident severity and causes
- Study the impact of weather, road type, and traffic density
- Discover temporal and geographical accident trends
- Generate meaningful insights through data visualization

---

## 📂 Dataset Information

- **Dataset:** Indian Road Accident Dataset
- **Records:** 20,000
- **Features:** 24
- **Time Period:** 2022–2025

---

---

## 📈 Key Insights

- 55.1% of accidents were classified as **Minor**.
- 29.9% were **Major** accidents.
- 14.9% were **Fatal** accidents.
- Driver distraction, overspeeding, weather, drunk driving, and poor road conditions contributed almost equally.
- Maharashtra recorded the highest number of accidents.
- Accident frequency increased during daytime and evening traffic hours.
- Weather showed minimal impact on accident severity.
